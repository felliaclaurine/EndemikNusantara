# 🌿 Endemik Nusantara

Aplikasi edukasi berbasis Android yang menampilkan informasi flora dan fauna endemik Indonesia.

## Tentang Aplikasi
Endemik Nusantara adalah aplikasi mobile yang menyajikan informasi lengkap mengenai hewan dan tumbuhan endemik Indonesia, dilengkapi dengan fitur pencarian, favorit dan dark mode.

## Fitur
- 🦜 Daftar hewan endemik Indonesia
- 🌺 Daftar tumbuhan endemik Indonesia
- 🔍 Pencarian real-time
- ⭐ Simpan spesies favorit
- 🌙 Dark mode & light mode

## Teknologi
- Java (Android)
- Material Design 3
- RecyclerView & GridLayout
- SharedPreferences
- Glide Image Loading
- Gson
